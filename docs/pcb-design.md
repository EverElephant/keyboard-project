# xboard73 PCB 设计说明

## 概述

73 键客制化机械键盘，nRF52840 主控，纯 BLE 无线（2×AA 电池供电）。

## 系统框图

```
┌──────────────────────────────────────────────────────────┐
│                     nRF52840-QIAA-R                       │
│                                                          │
│  P0.05-P0.11 ────→ 行输入（7 行，内置下拉）               │
│  P1.00-P1.10 ────→ 列输出（11 列）                        │
│                                                          │
│  P0.02 (AIN0) ────→ 电池电压检测（ADC，电阻分压）         │
│  P0.13         ────→ 电源开关控制（MOSFET 使能）          │
│  P0.15         ────→ 蓝牙状态指示灯（LED）                │
│  P0.16         ────→ 低电量指示灯（LED）                  │
│  P0.17 (RESET) ────→ 按键复位（外部拉高 + 按键到地）      │
│                                                          │
│  SWDCLK (P0.25) ───→ SWD 烧录接口                        │
│  SWDIO  (P0.26) ───→ SWD 烧录接口                        │
│                                                          │
│  32.768kHz │──→ 低速晶振（BLE 唤醒定时器）                │
│  32MHz     │──→ 高速晶振（射频）                           │
└──────────────────────────────────────────────────────────┘
```

## 电源设计

### 电池供电（纯无线）

- 2×AA 串联：标称 3.0V，工作范围 2.0V–3.2V
- **不需要升压/降压**：nRF52840 VDD 范围 1.7V–5.5V，直驱
- 电池正极串联 **10Ω 限流电阻** 后再进 VDD
- 靠近 VDD 引脚：**4.7µF + 0.1µF** 去耦电容

### 电源开关

- 使用 P-MOSFET（SI2301）做高侧开关
  - Gate 接 P0.13（低电平开启）
  - 10kΩ 上拉电阻到电池正极（确保默认关闭）
  - 按住 Esc 时 P0.13 输出低电平开启电源 → 实现按键开机
  - 固件软件关机：P0.13 输出高电平关闭 PMOS

### 电池电压检测

- 电池正极 → 1MΩ + 100kΩ 分压 → P0.02 (AIN0)
  - 分压比 = 100/(1000+100) = 1/11
  - 电池满电 3.2V → ADC 输入 0.29V
  - 电池低压 2.0V → ADC 输入 0.18V
  - nRF52840 ADC 参考可使用内部 0.6V，精度够
- 分压后并联 0.1µF 滤波电容到地

## 晶振

### 32MHz 主晶振（射频必需）
- 负载电容 12pF，推荐 NX3225GD-32.000MHZ
- 匹配电容：C_load = 12pF → 选 18pF 各一个（含 PCB 寄生）
- 走线：差分对、等长、远离天线区域

### 32.768kHz 低速晶振（可选，省电用）
- 用于 BLE 睡眠模式下精确定时唤醒
- 负载电容 6pF，选 12pF 对地电容

## 天线设计

**这是最关键的 RF 部分，上一版 nRF52832 不稳定最可能的问题就在这里。**

### 倒 F PCB 天线（推荐）
- 走线阻抗严格 **50Ω**
- 天线区域 **TOP/GND 层镂空**，不铺铜
- π 型匹配网络预留：串联 0Ω + 并联 NC（调试时换电容电感调匹配）
- 天线参数（设计参考 Nordic nRF52 参考设计）：
  - L 约 15.5mm（1.6mm FR4）
  - 天线净空区 ≥ 5mm
- **天线区域正下方 PCB 所有层不铺地**

### 如果选陶瓷天线（更稳定但贵）
- 推荐 ANT3216LL12R2400A（¥2-3 一颗）
- 同样需要 50Ω 走线和净空区
- 严格按 datasheet 的匹配电路

## PCB 层叠（推荐）

- 4 层板（成本略高但信号完整好）：
  - Top：信号 + 天线
  - L2：完整地平面（GND）
  - L3：电源（VDD 铺铜）
  - Bottom：信号
- 2 层板（省成本）：
  - Top：信号 + 天线
  - Bottom：地平面（尽可能连续铺铜）
  - 需要仔细走线避免地回路

## PCB 设计检查清单（EasyEDA 画板时注意）

- [ ] nRF52840 QFN 底部中心散热焊盘（GND）必须过孔到底层
- [ ] 32MHz 晶振靠近芯片，走线短，远离天线
- [ ] 去耦电容靠近 VDD 引脚，0.1µF 和 4.7µF 各一个
- [ ] 天线 50Ω 走线（参考微带线计算宽度：FR4 1.6mm 板 ≈ 0.3mm 线宽）
- [ ] 天线区域 PCB 顶层/内层全部镂空不铺铜
- [ ] π 型匹配网络靠近天线馈点放置
- [ ] 电池走线 > 0.5mm 宽
- [ ] USB 不接（纯无线设计），D+/D- 悬空或接地
- [ ] SWD 烧录口留好（5 pin + GND）
- [ ] 按键矩阵二极管方向统一
- [ ] 复位按键：外部 10kΩ 上拉到 VDD
- [ ] 未使用的 GPIO 悬空即可（nRF 内部默认禁用）
